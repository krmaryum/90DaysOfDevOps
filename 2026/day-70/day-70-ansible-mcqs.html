Day 70 - Ansible MCQs

Variables, Facts, Conditionals, Loops, Register and Debug

1. What is the main purpose of variables in Ansible?

Correct answer: B — Variables make playbooks flexible and reusable.

2. Which syntax is used to reference a variable in Ansible?

Correct answer: B — Ansible uses Jinja syntax like {{ variable }}.

3. Which variable source has the highest precedence?

Correct answer: C — Extra vars passed with -e override all others.

4. What does group_vars/all.yml apply to?

Correct answer: C — group_vars/all.yml applies to all hosts.

5. What does host_vars/worker-ubuntu.yml apply to?

Correct answer: B — host_vars files apply to the matching hostname.

6. If group_vars defines max_connections: 1000 and host_vars defines max_connections: 2000, which value is used?

Correct answer: B — host_vars override group_vars.

7. What module gathers Ansible facts from a host?

Correct answer: B — The setup module gathers facts.

8. Which fact is useful for selecting apt or yum?

Correct answer: A — os_family helps identify Debian or RedHat families.

9. Which fact usually contains the private IP address?

Correct answer: B — default_ipv4.address usually shows the private IP.

10. What does inventory_hostname represent?

Correct answer: B — inventory_hostname is the host name from inventory.

11. What is group_names?

Correct answer: A — group_names contains all groups for the current host.

12. Which keyword controls whether a task runs?

Correct answer: C — when controls conditional execution.

13. Which is the correct syntax for multiple AND conditions?

Correct answer: D — Multiple AND conditions can be written as a list under one when.

14. What happens if a when condition is false?

Correct answer: B — Ansible skips tasks when the condition is false.

15. What does loop do?

Correct answer: A — loop repeats the task for each item.

16. What variable represents the current item inside a loop?

Correct answer: C — item is the current loop value.

17. What is the modern replacement for with_items?

Correct answer: B — loop is the modern recommended syntax.

18. Which keyword stores output from a task?

Correct answer: C — register stores task results.

19. Which property contains normal command output from a registered result?

Correct answer: B — stdout contains the command output.

20. What does stdout_lines provide?

Correct answer: A — stdout_lines gives output as a list.

21. Why use changed_when: false for report commands?

Correct answer: B — Read-only commands should not be marked changed.

22. Which module can save a report file on a remote server?

Correct answer: A — The copy module can write content to a file.

23. What does the default() filter help with?

Correct answer: B — default() prevents failures when values are missing.

24. In YAML, what does > do in a multi-line msg?

Correct answer: B — > is a folded block scalar.

25. What is the biggest benefit of Day 70 concepts together?

Correct answer: B — Variables, facts, conditionals and loops create adaptive automation.
